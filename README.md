# Smart EV Charging Box - Enclosure V2

This repository contains the latest CAD iteration for the Smart EV Charging Box. This version focuses on optimization for space, component integration, and a more compact form factor.

## Overview
The latest iteration of the 3D design has been successfully built. We have significantly reduced the maximum internal air space and shrunk the overall box shape to make the device more compact than previous versions.

## Key Design Improvements

* **Compact Form Factor:** The enclosure has been resized to eliminate unnecessary volume.
* **Detection Switch Integration:** Includes dedicated mounting for a detection switch.
* **Smart Card Storage:** Features a new, built-in pouch designed to hold the car charger smart card securely.
